Kerbal Space Program - Rocket Flight Control using kRPC 
========================================================

Dependencies
------------

```bash
$ pip install zcm
$ pip install krpc
```

KSP Craft File: ZCM Rocket.craft

Run Experiment
--------------

```bash
$ zcm --config configuration.json
```
